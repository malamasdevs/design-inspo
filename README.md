# Design Inspo

Open source website templates for creative portfolios and studios. Each template is a complete single-page site for a fictional company with navigation, hero, work section, about, contact, and footer.

Built with Next.js, GSAP, and Canvas 2D. No paid dependencies.

## Templates

### 01 — Atelier Lumière
A minimal design studio. Warm cream palette, particle typography hero with curl-noise flow field, editorial project list, and dark about section.

### 02 — Vantage
A venture studio. Dark editorial layout with scroll-driven stacked portfolio cards. Each card is a full-viewport case study with funding data and key metrics.

### 03 — Ren Ishikawa
A creative director portfolio. Bold dark aesthetic with 3D magnetic tilt gallery, oversized type, and experimental hover effects.

## Quick Start

```bash
git clone https://github.com/malamasdevs/design-inspo.git
cd design-inspo
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js (App Router)
- GSAP + ScrollTrigger
- Canvas 2D with Perlin noise
- CSS 3D transforms
